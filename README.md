# Placement Preparation Dashboard

A web-based dashboard for tracking placement preparation progress.

## Features

- Track DSA problems solved
- Track aptitude questions completed
- Track certifications earned
- Persistent storage using Local Storage
- Progress visualization

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Future Enhancements

- Dark Mode
- Analytics Dashboard
- User Authentication
- Cloud Database Integration
